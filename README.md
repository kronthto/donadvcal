# DON-Adv.-Cal - Win checker for Donauwoerths advent calendar

[![Software License][ico-license]](LICENSE.md)

Checks whether a set of calendar numbers is in the Lions Club daily price drawing and informs via E-Mail.

## Install / Use

``` bash
$ composer install
$ cp .env.example .env
$ ./check-don-advcal
```
